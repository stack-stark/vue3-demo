# vue3构建新的中后台

## 启动
`要使用yarn安装依赖,npm安装会报错无法启动`

1.使用`yarn install` 命令安装npm包

2.运行`yarn serve`

3.打开`localhost:8080`;

## 开发时代理修改
修改`vue.config.js`中的`devServer`的`proxy`
